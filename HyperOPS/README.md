## 🛠 How to Use This Project

### 1. The Setup
Open this folder in an Agentic IDE (Cursor, Windsurf, or similar). The Agent will automatically detect `AGENT_INSTRUCTIONS.md` and understand its role as your **Elite Hyperliquid Trader**.

### 2. The Workflow (Daily Trading Routine)
This project uses a strict **Command-Response** workflow.

#### Step 1: Initialize
Start every session by commanding:
> *"Read Handbook"*

The Agent will read the `Agent Handbook/` to load its instructions and protocols.

#### Step 2: Execute Routine
Choose your mode:

**Option A: Standard Daily Run**
> *"Run the Daily"*
Agent follows `daily_OPS.md` for standard analysis and execution.

**Option B: HyperGrok Mode (X-Optimized)**
> *"HyperGrok Run the Daily"*
Agent follows `daily_OPS.md` enhanced by `HyperGrok_Prompt.md` for real-time X alpha mining.

#### Step 3: Execution
The Agent will generate a **Daily Trading Brief**.
**You must explicitly command:**
> *"Execute all trades"* or *"Execute trade [X]"*

### 3. The Outputs
*   **Daily Trading Briefs:** Comprehensive market analysis with actionable trade setups.
*   **Knowledge Graph:** A persistent database of trading intelligence:
    *   `tokens.md`: Hyperliquid perpetuals characteristics and setups.
    *   `narratives.md`: Market regimes and catalyst themes.
    *   `smart_money.md`: Elite trader profiles and whale activity.
    *   `playbook.md`: Proven setups with success rates.
    *   `performance_log.md`: Detailed PnL tracking and strategy optimization.
*   **Trade Analysis:** Deep dives into each trade for continuous improvement.

3. **Execution Phase**: **Instruct the agent** to execute specific trades via API
